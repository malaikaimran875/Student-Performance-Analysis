# Student Performance Analysis

## Project Overview

This project analyzes student academic performance using Python, Pandas, and Matplotlib. The goal is to identify patterns in student scores and understand factors that affect academic performance.

## Tools Used

* Python
* Pandas
* Matplotlib
* VS Code

## Dataset

The dataset contains information about student demographics and exam scores in:

* Math
* Reading
* Writing

## Analysis Performed

* Dataset exploration and summary statistics
* Average score comparison across subjects
* Gender-wise performance analysis
* Impact of test preparation courses
* Score distribution visualization
